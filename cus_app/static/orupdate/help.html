Target Parameter Update Status Help Page

If there are multiple revisions of the same obsid on the list:

  1. Different revisions of the same obsid are indicated by the same color marker on the "OBSID.revision" column.
  2. If "Multiple Revisions Open" is listed in the "Note column," the obsid has multiple revisions that are still open. Please verify all earlier revisions before verifying the latest one.
  3. If "Higher Rev # Was Already Signed Off" is listed in the "Note column," the obsid has a newer entry already signed off. Please verify all versions of the obsid on the table, as the latest one (which was already signed off) supersedes all previous revisions. If an earlier version of the obsid contains unsigned edits which are no longer desired, consult the "Discard" button.
  4. If "Large Coordinate Shift" is listed in the "Note column," then the Right Ascension and Declination coordinates were collectively shifter by more than 8 arc minutes.
  5. If "New Comment" is listed in the "Note column," then a new comment has been listed with relevant details to the revision.
    6. If "On Approved List" is listed in the "Note column," then this Obsid is already approved by Usint and so that approval is assumed to still be given once the revision is completed.
  6. You can clear any "out-of-date" entries by clicking the "Discard" button, even if the general, ACIS, ACIS SI Mode, or HRC SI Mode edits columns are not signed off yet. Any of the unsigned column entries will be recorded as "N/A" in the database.
